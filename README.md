# TaskMaster

A basic To-Do list app that I created to study for Google's Associated Android Developer Certification exam. (Kotlin Version)

The app covers the following topics: 

- Room Database Implementation (Including Entity, Dao, RoomDatabase)
- Repository Implementation
- ViewModel and ViewModel Factory Implementation (Including LiveData and Coroutines)
- Navigation Drawer Implementation (Including Navigation Component library)
- RecyclerView Implementation (Including List Adapter and DiffUtil.ItemCallback)
- AppBarLayout with Toolbar Implementation
- ViewPager Implementation
- Custom View Implementation
- Shared Preferences Implementation 
- App Theme and Styles Implementation (Including Dark/Light Theme)
- Alarm Manager
- Notifications (With Broadcast Receiver)
